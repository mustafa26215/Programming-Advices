1) What is literal?
A literal is a fixed value written directly in the program code. It represents a constant value that does not change during program execution (like 10, 'A', "Hello").

2) Can you assign different values to a literal?
No. A literal is a constant value, so you cannot change or assign different values to it.

3) What are literal types?
Common literal types include:

Integer literals → 10, 25, -5

Floating-point literals → 3.14, 2.5

Character literals → 'A', 'b'

String literals → "Hello"

Boolean literals → true, false

4) Why do we need escape sequence literals?
Escape sequences are used to represent special characters in strings or characters that cannot be typed directly, such as a new line (\n), tab (\t), or quotation marks (\"). They help control formatting and display special symbols in output.
