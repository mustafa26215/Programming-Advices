Are the following valid variable names and why?

Age → ✅ Valid (starts with a letter, no special characters).

My-Age → ❌ Invalid (contains -, not allowed).

My_Age → ✅ Valid (underscore _ is allowed).

_Age → ✅ Valid (can start with underscore).

2x → ❌ Invalid (cannot start with a number).

X2 → ✅ Valid (starts with a letter).

My Name → ❌ Invalid (contains a space).

MyName → ✅ Valid (no spaces or special characters).

#Name → ❌ Invalid (# not allowed).

$Name → ❌ Invalid ($ not allowed in C++).

N@me → ❌ Invalid (@ not allowed).

int → ❌ Invalid (reserved keyword).

float → ❌ Invalid (reserved keyword).
